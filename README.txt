開發者：jimmychen（u/jp6）

📌 工具簡介：
阿孟萬能連點器是一款多功能的滑鼠與鍵盤自動點擊工具，支援自訂熱鍵、點擊間隔、多段腳本、自動移動、滑鼠座標記錄、腳本儲存與匯入等進階功能。
非常適合用於遊戲輔助、自動化流程、測試操作等場景。

✅ 功能特色：
- 熱鍵啟動與停止（預設 F6）
- 自訂點擊間隔與隨機延遲
- 點擊次數上限與延遲啟動
- 滑鼠與鍵盤自動操作腳本
- 多段動作錄製與儲存/讀取
- 即時操作日誌
- 支援暗黑主題 UI
- 支援 Windows 平台（Python 打包）

🛠️ 如何使用：
1. 執行「阿孟萬能連點器.exe」
2. 設定所需參數與動作（加入滑鼠位置、鍵盤操作）
3. 點擊「開始 / 停止」或按下指定熱鍵開始自動執行
4. 可儲存腳本為 JSON，日後直接載入使用
